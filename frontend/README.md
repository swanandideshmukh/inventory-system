# 📦 Smart Inventory & Supply Chain Management System

## 🌐 Live Demo
👉 [Click here to view](https://inventory-system.vercel.app)

## Features
- 📦 Stock level monitoring with real-time alerts
- 📉 Demand forecasting
- 🏭 Supplier management
- 🔔 Automated low-stock alerts via Redis
- 📄 Exportable reports

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Cache/Alerts:** Redis
- **Deployment:** Vercel + Render
